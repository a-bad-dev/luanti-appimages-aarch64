# luanti-appimages-aarch64

AppImages of [Luanti](https://luanti.org/) for Linux on aarch64, such as modern Raspberry Pis.

The compilation bash script was largely taken from https://github.com/rollerozxa/luanti-appimage and modified for aarch64.

This script can be run directly (as root) to generate an AppImage in the current working directory, unlike the upstream version which needs GitHub thingys to run.
